# 🤖 Анонимный чат в Telegram

## 📌 Описание

Данный проект представляет собой Telegram-бота, который позволяет пользователям анонимно общаться друг с другом. После команды `/start` бот автоматически подбирает случайного собеседника и пересылает сообщения между участниками, не раскрывая их личные данные.

---

## 🛠️ Технологии

- Python
- Pyrogram
- Telegram Bot API

---

## ✨ Функционал

- Поиск случайного собеседника.
- Создание анонимного чата между двумя пользователями.
- Автоматическое соединение пользователей в пары.
- Пересылка текстовых сообщений между собеседниками.
- Простая и понятная логика работы.

## 🚀 Запуск проекта

1. Установите необходимые библиотеки:

```bash
pip install -r requirements.txt
```

2. Укажите данные Telegram API и токен бота в файле `config.py`:

```python
API_ID = ...
API_HASH = "..."
BOT_TOKEN = "..."
```

3. Запустите программу:

```bash
python main.py
```

---

## ⚙️ Принцип работы

1. Пользователь отправляет команду `/start`.
2. Если другой пользователь уже ожидает, бот соединяет их в анонимный чат.
3. Если свободного собеседника нет, пользователь добавляется в очередь ожидания.
4. После соединения все сообщения автоматически пересылаются между участниками.

Проект создан для изучения разработки Telegram-ботов с использованием библиотеки Pyrogram, обработки сообщений пользователей и организации анонимного общения между ними.
